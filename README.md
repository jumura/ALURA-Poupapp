# 💸 Poupapp - Projeto Pessoal usando TS e React
O Poupapp é um aplicativo para acompanhar as suas despesas financeiras, mostrando quanto você ganhou e gastou, em quais categorias seus gastos se concentram mais e o seu orçamento diário disponível!
<br><br>
> O objetivo deste projeto é aprender mais a fundo sobre TS com React. O design já veio feito, graças a **Alura**, apenas alterei a lógica por trás do aplicativo, de acordo com o curso *"React com Typescript: aplique Hooks e Context API tipados"*.

## 📝 Maiores aprendizados durante o curso
- Hooks do React, como *useRef* e *useImperativeHandler*, e suas importâncias ao lidar com elementos DOM, evitar prop drilling e melhorar a performance de SPAs.
- *Context API* e a sua importância para disponibilizar os dados para **qualquer** componente que queira utiliza-los, criando um "contexto" onde não é necessário passar de componente para componente.
- Utilitários do TypeScript, como *Omit*, *Pick*, *Partial* e *Record*, que ajudam a manipular tipos de maneira **flexível**.
- Hooks avançados, como *useMemo*, para otimização de perfomance dos aplicativos.

## ⚙️ Principais dependências utilizadas
- axios
- json-server
- react-router
- recharts
 
## 🧭 Guia de como rodar o projeto
1. Clone o repositório na sua máquina
2. Instale as dependências
3. Crie um arquivo chamado *"db.json"* na pasta principal do projeto, assim salvando o nome e renda do usuário
4. Abra dois terminais: no 1° digite *"npm run dev"* e no 2° *"npm run server"*
5. Acesse o *"http://localhost:5173/"* e pronto!
